Introduction

The purpose of this project is to build a robot that will track and follow a ball of a specified color. In order to achieve this goal, basic principles of computer vision and control systems are applied to detect, follow and correct the path of the robot as it approaches the pre-determined object. The robot is powered by a Raspberry Pi and Arduino Nano. The Raspberry Pi uses a webcam to capture video and applies a set of filters using OpenCV library to detect and track the location of the ball. The data is transferred to the Arduino Nano, which receives the data and uses a closed loop feedback to control the robot.

OpenCV

Computers cannot process images like humans do. There are many methods for getting a computer to recognize people, objects, and different colors. OpenCV offers a large library for image processing in python. For this project, any color ball can be used. However, it is easier to detect a ball whose color stands out from its background. There are a set of filters that are applied below to detect the ball:

  1. The first filter sets the image to an HSV scale, which stands for Hue, Saturation and Value. Each color has a different representation for those 3 values. For example, a green object can have the HSV set as (0,128,0) where H is 0, S is 128 and V is 0.
  2. Once the HSV values are determined, the values are used in the program to isolate the determined color from the rest of the image. A filter is applied to convert the image to black and white.
  3. Once an image is black and white, the computer will run through each individual pixel in the image and determine which pixels contain the object. The object will be denoted by white and the background will be black. Once the computer has processed the image, it will draw a circle based on the edges of the ball left by the black and white image.
  4. The computer will obtain the center of the by using the equation of a circle and solving for both variables for the vertical and horizontal position. Since horizontal position of the ball is needed, that is the variable that will be used in the program.
